My first post

8/1/2018 In Random Thoughts

Yo!

So I made this a new website/portfolio. I was growing dissatisfied with my old website, so I knew it was time for an upgrade. Plus my skills have greatly advanced since I made my first website/portfolio back in 2014.

I started coding websites in 2014. It was really just a way for me to stay busy at work. (I had a lot of downtime!) HTML and CSS were pretty easy to learn. Picking up javascript was a bit more challenging, but it toke my web development skills to a new level. To be honest, I still don’t fully understand javascript, but I understand enough to get by.

By no means would not I consider myself an expert web developer. But I can hold my own with front-end web development work. To be honest, it’s kind of fun for me. So let's just say front-end web development is my hobby.

For this site in particular, I bought an HTML template on ThemeForest for like 10 bucks. So I would not have to code this bad boy from scratch. I got icons some from flaticon to help support the design.

But, whatever, this is my new site I hope you like it. I’ll be posting random thoughts and ideas here from time to time. I’ll try to keep it professional and write about digital marketing insights and stuff. But we are not defined by how we pay our bills. So I also plan to share my perspectives on new music and current events. We’ll see how it goes. So follow along or not. Whatevers clever!
